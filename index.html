⚡ Get Your Free Groq API Key

Takes about 2 minutes. No credit card required.

1
OPEN GROQ CONSOLE
Go to console.groq.com → click "Sign Up" (Google login works).
2
API KEYS SIDEBAR
Click "API Keys" in the left sidebar.
3
CREATE & COPY
Click "Create API Key", name it anything, click Submit. Copy the key starting with gsk_ — you only see it once!

⭐ Get Your Gemini API Key

Via Google AI Studio. Free tier available in most countries.

1
OPEN GOOGLE AI STUDIO
Go to aistudio.google.com and sign in with Google.
2
GET API KEY
Click "Get API Key" in the sidebar → "Create API key" → select or create a project.
3
COPY YOUR KEY
Copy the key — it starts with AIza.
⚠ Google restricts free Gemini API access in some regions. If you get a quota error, switch to Groq instead.

🤖 Get an OpenAI API Key

Requires a paid account. Minimum $5 credit to get started.

1
CREATE OPENAI ACCOUNT
Go to platform.openai.com and sign up or log in.
2
ADD BILLING CREDIT
Go to Settings → Billing, add a card, and load at least $5 in credits.
3
CREATE API KEY
Click "API Keys" in the sidebar → "Create new secret key". Copy it immediately — starts with sk-.
💡 GPT-4o costs ~$0.01–$0.03 per conversation. Your $5 gives you hundreds of history conversations.
